# vcqi_jamaica_mics_2022_manuscript
Materials to accompany the manuscript on secondary analysis of the 2022 UNICEF Multi-Indicator Cluster Survey (MICS) in Jamaica using the World Health Organization (WHO) and Pan American Health Organization (PAHO) software named Vaccination Coverage Quality Indicators (VCQI).

The folders include:

1. Programs to make the Jamaica MICS data compatible with VCQI - both Stata and R versions.
2. Control programs to run VCQI to replicate the Jamaica analyses - both Stata and R versions.
3. A snapshot of the Stata VCQI sourcecode used to analyze the Jamaica data. (includes VCQI documentation)
